Flex 布局

容器默认存在两根轴，水平(main axis) 和 垂直(cross axis)

水平轴起止位置 main start 、 main end

垂直轴起止位置 cross start 、 cross end

容器属性

  • flex-direction
    • 决定main轴方向
  • flex-wrap
    • 轴线上的换行
  • flex-flow
    • flex-direction & flex-wrap 的简写
  • justify-content
    • main 轴对齐方式
  • align-items
    • cross 轴对齐方式
  • align-content
    • 多根轴线的对齐方式。单个轴时无效

项目属性

  • order
    • 项目排序
  • flex-grow
    • 放大比例（默认0，空间剩余时，不自动放大）
  • flex-shrink
    • 缩小比例（默认1，空间不足时，自动缩小）
  • flex-batis
    • 分配多余空间之前 item 占 main 轴的空间（设置长度将占据固定空间，默认 auto）
  • flex
    • flex-grow & flwx-shrink & flex-basis 的缩写（默认 0 1 auto。 auto 等于 1 1 auto。 none 等于 0 0 auto）
  • align-self
    • 允许单个 item 与其他 item 对齐方式不同，可覆盖 align-items。（默认 auto 表示继承父元素或 stretch）
